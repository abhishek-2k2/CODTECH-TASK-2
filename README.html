News Website

Overview

This project is a dynamic news website developed using HTML, CSS, and JavaScript. It aggregates real-time news from various sources, providing users with up-to-date information on different topics. The website features a search function and category-based filtering to enhance content discovery and user experience.

Features

  • Real-Time News Aggregation: Fetches and displays news articles from various sources using an external API.
  • Search Functionality: Allows users to search for news articles based on keywords.
  • Category-Based Filtering: Users can filter news by categories like IPL, Finance, and Politics.
  • Responsive Design: Ensures a smooth experience across different devices and screen sizes.

Technologies Used

  • HTML: Structure and layout of the website.
  • CSS: Styling and design for a visually appealing interface.
  • JavaScript: Dynamic functionality, including API integration and user interaction handling.

Getting Started

  1. Clone the Repository: 
    git clone https://github.com/your-username/news-website.git
  2. Navigate to the Project Directory: 
    cd news-website
  3. Open index.html: Open index.html in your web browser to view the website.

API Key

The project uses the News API. You need an API key to fetch news articles. Replace the placeholder in script.js with your API key:

const API_KEY = "YOUR_API_KEY_HERE";

File Structure

  • index.html: Main HTML file containing the structure of the website.
  • style.css: CSS file for styling the website.
  • script.js: JavaScript file for handling API requests and dynamic content.
